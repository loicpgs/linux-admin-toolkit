# Linux Admin Toolkit

## Overview
This project is a Linux administration toolkit built with Bash scripts and versioned using Git.

The goal is to provide simple and reusable scripts to manage and monitor a Linux system.

## Objectives
- Automate common system administration tasks
- Practice Bash scripting
- Use Git for version control with clean commits
- Build a professional project for portfolio and interviews

## Features (planned)
- System update automation
- Disk usage monitoring
- Service status checks
- Backup scripts
- Log management
- Network diagnostics

## Project Structure

linux-admin-toolkit/
├── scripts/     # Bash scripts
├── docs/        # Documentation
├── logs/        # Log files
├── README.md
└── .gitignore

## Requirements
- Linux (Debian/Ubuntu recommended)
- Bash
- Git
